# 成果展示与总结

## 学习目标

- 能清楚介绍自己的作品
- 会从反馈中找出下一步优化方向
- 总结自己已经掌握的能力

## 展示时可以说什么

1. 我做了什么
2. 这个作品解决什么问题
3. 最满意的一个功能是什么
4. 开发时遇到的最大困难是什么
5. 如果继续做，我还想加什么

## 复盘问题

- 哪个 Prompt 最有效
- 哪一步最容易出错
- 你最想继续深入哪种项目

## 下一步方向

- 继续做更完整的网站
- 尝试小游戏合集
- 给自己的学习流程做一个管理工具
- 学习基础 Git 和版本管理

## 小结

作品、反馈、迭代，才是真正的成长闭环。会展示、会复盘、会继续改，比“只做完一次”更重要。
