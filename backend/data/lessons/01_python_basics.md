# Python基础

## 学习目标

- 认识 Python 的特点与常见应用场景
- 理解变量、基本数据类型与输入输出
- 会写最简单的顺序程序

## Python 是什么

Python 是一种语法简洁、可读性强的编程语言。它常被用于数据分析、自动化脚本、网站后端、人工智能和教学入门。

## 第一个程序

```python
print("Hello, Python!")
```

`print()` 用来把内容输出到屏幕。

## 变量与数据类型

```python
name = "Alice"
age = 12
height = 1.55
is_student = True
```

- `str`：字符串
- `int`：整数
- `float`：小数
- `bool`：布尔值，只有 `True` 和 `False`

## 输入与输出

```python
name = input("请输入你的名字：")
print("你好，" + name)
```

`input()` 读入的内容默认是字符串。如果要计算，需要先转换类型。

```python
age = int(input("请输入年龄："))
print(age + 1)
```

## 常用运算符

```python
print(3 + 2)
print(10 / 2)
print(7 % 3)
print(2 ** 3)
```

## 常见错误

- 忘记给字符串加引号
- 用 `input()` 读到的是字符串，却直接参与数值计算
- 缩进不统一
