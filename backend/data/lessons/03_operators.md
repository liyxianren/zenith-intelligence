# 运算符与表达式

## 学习目标

- 掌握算术、关系、逻辑运算符
- 理解自增自减
- 初步认识位运算

## 算术运算

```c
int a = 10, b = 3;
printf("%d\n", a + b);
printf("%d\n", a - b);
printf("%d\n", a * b);
printf("%d\n", a / b);
printf("%d\n", a % b);
```

## 关系运算

```c
printf("%d\n", a > b);
printf("%d\n", a == b);
```

在 C 中，条件成立通常视为非 0，不成立为 0。

## 逻辑运算

- `&&`：并且
- `||`：或者
- `!`：取反

```c
if (a > 0 && b > 0) {
    printf("都大于 0\n");
}
```

## 自增与自减

```c
int x = 5;
printf("%d\n", x++);
printf("%d\n", ++x);
```

## 位运算

- `&`：按位与
- `|`：按位或
- `^`：按位异或
- `<<`：左移
- `>>`：右移

## 小结

表达式由变量、常量和运算符构成。理解运算符优先级，可以减少很多难查的 bug。
