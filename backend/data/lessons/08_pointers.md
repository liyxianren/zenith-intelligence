# 指针基础

## 学习目标

- 理解地址与指针的概念
- 会使用 `&` 和 `*`
- 理解指针与数组的关系

## 什么是指针

指针变量保存的是内存地址。

```c
int num = 10;
int *p = &num;
```

- `&num`：取地址
- `*p`：解引用，访问地址中的值

## 基本操作

```c
printf("%d\n", num);
printf("%p\n", (void *)&num);
printf("%d\n", *p);
```

## 通过指针修改值

```c
*p = 20;
printf("%d\n", num);
```

## 指针与数组

数组名在很多表达式中会退化为首元素地址。

```c
int arr[3] = {1, 2, 3};
int *p = arr;
printf("%d\n", *(p + 1));
```

## 指针作为函数参数

```c
void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}
```

## 小结

指针是 C 语言的核心主题。只要牢牢记住“指针存地址，`*` 取值，`&` 取地址”，就能逐步建立直觉。
