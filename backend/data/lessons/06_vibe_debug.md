# 当 AI 出错了怎么办

## 学习目标

- 认识常见错误类型
- 学会最基本的排错流程
- 会向 AI 提供有效报错信息

## 三步排错法

1. 复现问题：稳定地让错误再次出现
2. 缩小范围：判断是页面结构、样式还是逻辑问题
3. 带证据提问：把报错信息和相关代码一起给 AI

## 高质量提问示例

```text
这段 JavaScript 运行后按钮点击没有反应。
报错信息是：
Uncaught TypeError: Cannot read properties of null

请你：
1. 先解释报错含义
2. 指出最可能的原因
3. 给出最小修改方案
```

## 常见错误

- 元素选择器写错
- JS 在 DOM 加载前执行
- 变量名拼写不一致
- 路径不对导致文件没加载到

## 小结

调试不是“猜”，而是“缩小范围”。只要把症状、位置和预期讲清楚，AI 才能帮你更快定位问题。
