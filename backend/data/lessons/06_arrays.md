# 数组

## 学习目标

- 会声明和访问一维、二维数组
- 理解数组下标从 0 开始
- 初步了解字符串在 C 中的表示方式

## 一维数组

```c
int scores[3] = {90, 85, 88};
printf("%d\n", scores[0]);
```

## 遍历数组

```c
for (int i = 0; i < 3; i++) {
    printf("%d\n", scores[i]);
}
```

## 二维数组

```c
int matrix[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};
```

## 字符串

C 语言的字符串本质上是字符数组，以 `'\0'` 结尾。

```c
char name[] = "Alice";
printf("%s\n", name);
```

## 常用字符串函数

- `strlen()`：求长度
- `strcpy()`：复制
- `strcmp()`：比较

使用这些函数前需要：

```c
#include <string.h>
```

## 小结

数组非常适合存放同类型的连续数据。理解数组和下标，是后续学习指针的基础。
