# 做一个属于你的小工具

## 学习目标

- 学会从真实需求出发设计小工具
- 会让 AI 按功能清单生成页面
- 理解“最小可用版本”的思路

## 工具题材建议

- 番茄钟
- 单词记忆卡
- 随机点名器
- 倒计时页面
- 每日打卡清单

## 写需求时要说清楚

- 工具解决什么问题
- 页面上有哪些按钮
- 需要保存哪些数据
- 用户完成一次操作后会看到什么

## 示例 Prompt

```text
请帮我做一个番茄钟网页工具。
要求：
1. 有开始、暂停、重置按钮
2. 默认 25 分钟
3. 倒计时结束后弹出提示
4. 样式简洁，适合学生使用
5. 用原生 HTML、CSS、JavaScript
```

## 小结

小工具比大项目更适合训练“提需求”和“快速迭代”的能力。先解决一个小问题，比一次做太多功能更稳。
