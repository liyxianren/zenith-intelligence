# 学会跟 AI 说话

## 学习目标

- 理解 Prompt 的作用
- 会写更清晰的需求
- 知道如何逐步逼近想要的结果

## 一个好 Prompt 的结构

可以先按下面的模板写：

```text
你是谁：
你要做什么：
输出格式：
限制条件：
示例：
```

## 对比

### 模糊写法

```text
帮我写个游戏。
```

### 清晰写法

```text
你是一个前端小游戏开发助手。
请用 HTML、CSS、JavaScript 写一个贪吃蛇游戏。
要求：
1. 画布大小 400x400
2. 使用方向键控制
3. 页面显示当前分数
4. 所有代码放在一个 HTML 文件中
```

## 追问技巧

- “请把代码拆成 HTML、CSS、JS 三部分解释。”
- “请只保留初学者能看懂的版本。”
- “请先指出 3 个可能出错的地方。”

## 小结

Prompt 不是一次写完的。好的结果通常来自“先描述，再观察，再追问”。
