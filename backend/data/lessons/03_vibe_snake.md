# 做一个贪吃蛇游戏

## 学习目标

- 用 AI 生成第一个可玩的作品
- 理解网页三要素：HTML、CSS、JavaScript
- 学会做第一轮功能迭代

## 推荐 Prompt

```text
你是一个前端小游戏开发工程师。
请用 HTML + CSS + JavaScript 生成一个贪吃蛇游戏。
要求：
1. 单文件实现
2. 方向键控制
3. 显示分数
4. 游戏结束后可以重新开始
5. 注释要写给初学者看
```

## 阅读代码时先看什么

1. 页面结构：画布和分数显示在哪里
2. 游戏状态：蛇、食物、方向、分数
3. 主循环：什么时候刷新画面
4. 输入处理：按键如何改变方向

## 第一轮改造建议

- 调快或调慢蛇的速度
- 修改颜色
- 添加最高分
- 增加暂停功能

## 小结

不要一开始就追求“完美代码”。先跑起来，再一轮轮改小功能，这是最有效的入门方式。
