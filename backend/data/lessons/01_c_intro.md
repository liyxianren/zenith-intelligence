# C语言入门

## 学习目标

- 了解 C 语言的用途和特点
- 会写并编译第一个程序
- 理解一个 C 程序的基本结构

## C 语言为什么重要

C 语言接近底层，执行效率高，常用于操作系统、嵌入式、编译器和高性能程序开发。学习 C 有助于理解内存、指针和程序运行机制。

## 第一个程序

```c
#include <stdio.h>

int main(void) {
    printf("Hello, C!\n");
    return 0;
}
```

## 程序结构说明

- `#include <stdio.h>`：引入标准输入输出库
- `main`：程序入口
- `printf`：输出内容
- `return 0`：表示程序正常结束

## 编译流程

1. 编写源文件，例如 `hello.c`
2. 编译生成可执行文件
3. 运行程序

示例命令：

```bash
gcc hello.c -o hello
./hello
```

## 常见错误

- 忘记写分号
- 花括号不匹配
- `main` 函数没有返回值

## 小结

C 语言语法比 Python 更严格，但也更能帮助你理解程序在计算机中是如何执行的。
